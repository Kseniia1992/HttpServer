Http-сервер на фреймворке netty (http://netty.io/), со следующим функционалом:

1. По запросу на http://somedomain/hello отдает «Hello World».

2. По запросу на http://somedomain/redirect?url=<url> происходит
переадресация на указанный url.

3. По запросу на http://somedomain/status выдается статистика:
 
 - общее количество запросов;
 - количество уникальных запросов (по одному на IP);
 - счетчик запросов на каждый IP: IP, кол-во запросов, время последнего запроса;
 - количество переадресаций по url'ам: url, кол-во переадресаций;
 - количество соединений, открытых в данный момент.

Инструментарий:

 Рабта с базой данных через JPA, ORM.
 Для запуска сервера необходимо создать базу данных PostgreSQL 
  - database: ServerDB
  - username: postgres 
  - password: qwerty

 Настройки подключения к базе данных в persistence.xml

 Язык программирования - Java 1.6

 Запуск сервера: java -jar HttpServer.jar.
